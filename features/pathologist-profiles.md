# Feature 2: Pathologist Profiles Showcase

## Objective
Highlight each speech-language pathologist with clear specialties and qualifications so patients can understand who they may work with.

## User Story
As a patient, I want to view therapist specialties and qualifications so I can feel confident about the clinic team.

## Required Profile Fields
- Full name
- Specialty
- Qualifications
- Optional short professional bio
- Optional photo

## UI Requirements
- Display profiles in a clean card layout.
- Keep text concise and easy to scan.
- Ensure specialty and qualifications are visually emphasized.
- Maintain readable spacing and contrast.

## Content Guidelines
- Specialty examples: pediatric language, post-stroke communication, dysphagia care.
- Qualification examples: M.S. SLP, CCC-SLP, state licensure.
- Keep profile copy plain-language and patient-friendly.

## Acceptance Criteria
1. Every displayed profile includes specialty and qualifications.
2. Layout is responsive and readable on mobile and desktop.
3. Cards have consistent structure and visual hierarchy.
4. No profile renders with missing required fields.

## Test Checklist
- Verify all profiles render with required fields.
- Verify long qualifications wrap cleanly without layout break.
- Verify profile section remains accessible via keyboard navigation.
- Verify images (if provided) include descriptive alt text.
