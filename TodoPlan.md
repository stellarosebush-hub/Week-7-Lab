# Todo File Plan

## Goal
Create a class-focused todo tracker with visible class names, due dates, and progress bars per item.

## Structure
- One table with columns:
  - Class Name
  - To Do Item
  - Due Date
  - Progress
  - Status
- Progress format:
  - [#####-----] 50%
- Date format:
  - YYYY-MM-DD

## Update Rules
1. Add new tasks as new rows.
2. Keep class names explicit and consistent.
3. Update progress bars and percentages together.
4. Mark status as:
   - Not Started
   - In Progress
   - Done

## Weekly Workflow
1. Review all due dates every Monday.
2. Move urgent tasks to the top of each class group.
3. Update each progress bar after study sessions.
4. Recalculate overall progress at the top.

## Visibility Requirements
- Class name must always be in the first column.
- Due date must always be visible in the third column.
- Every todo item must include a progress bar and percentage.
